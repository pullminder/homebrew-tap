# Homebrew Tap for Pullminder

Homebrew formulae for the [Pullminder CLI](https://github.com/pullminder/cli) — AI-powered PR analysis with risk scores, policy enforcement, and automated code review.

## Install

```sh
brew install pullminder/tap/pullminder
```

Verify the installation:

```sh
pullminder version
```

## Quick start

```sh
# Set up Pullminder in your project
pullminder init

# Analyze your current branch (works offline)
pullminder check

# Run in CI pipelines
pullminder ci

# Analyze a remote PR
pullminder diff https://github.com/org/repo/pull/42
```

The CLI includes 13 commands covering local analysis, remote PR scoring, rule pack management, git hooks, authentication, and custom registry authoring. Run `pullminder --help` for full usage.

## Update

```sh
brew upgrade pullminder
```

## Uninstall

```sh
brew uninstall pullminder
brew untap pullminder/tap
```
