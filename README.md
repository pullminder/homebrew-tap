# Homebrew Tap for Pullminder

## Install

```sh
brew install pullminder/tap/pullminder
```

## Update

```sh
brew upgrade pullminder
```

## About

This tap is automatically updated by [GoReleaser](https://goreleaser.com) when new CLI versions are released. The formula in `Formula/pullminder.rb` is auto-generated — do not edit manually.

For more information, visit [pullminder.com](https://pullminder.com).
